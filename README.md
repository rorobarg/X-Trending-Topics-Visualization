# Twitter Trending Topics Visualization

This project visualizes the top trending topics on Twitter (X) for the Philippines using the [Twitter Trends API](https://rapidapi.com/) via RapidAPI. It fetches live trending data and displays it in a horizontal bar chart using **Chart.js**.

---

## Features
- Fetches trending topics for the Philippines (WOEID: 23424934).
- Displays the top 25 topics with their tweet/post volumes.
- Interactive horizontal bar chart built with Chart.js.
- Error handling with a user-friendly message if the API fails.
- Dynamic date display on the page.

---

## Tech Stack
- **JavaScript (ES6)**  
- **Chart.js** for data visualization  
- **RapidAPI** for Twitter trends data  
- **HTML/CSS** for the frontend
